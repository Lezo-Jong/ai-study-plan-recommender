# Project Context

## Purpose

Generate personalized AI-powered study plans and adaptive learning recommendations using Gemini API.

## Security Rules

* Never expose API keys.
* Validate user inputs before processing.
* Do not store sensitive personal information.
* Prevent prompt injection where possible.

## Coding Standards

* Prefer modular functions.
* Separate UI logic from business logic.
* Maintain clear state transitions.
* Document major workflow changes.

## Evaluation Goals

* Accurate study plan generation.
* Reliable progress tracking.
* Adaptive recommendation quality.
* Consistent user experience.
